# QCENGINE

***

## Using QcEngine

First include the necessary include files, including the SDL headers in the project.

Then, link against the library as well as the SDL libraries.

Finally, add the following libs as additional dependencies

QcEngine.lib  
SDL2.lib  
SDL2main.lib  
SDL2_ttf.lib  
OpenAL32.lib  
Ws2_32.lib
